Common Array Algorithms

My Includes

checks if an array includes a specific things
click button and type one of the things in the list or dont to see if the list contains them (go to console for true/false)
list: '1', '2', '3', 'flowers', 'genuine', 'Maynards'
(does take capitals into account)

My IndexOf

checks if an array includes a specific things then states where those things are
click button and type one of the things in the list or dont to see if the list contains them (go to console to see if its there (and where) or not)
list: '1', '2', '3', 'flowers', 'genuine', 'Maynards'
(does take capitals into account)

Average Function Assignment

will average the numbers in the 'num' array

My Max Assignment

will find the greatest value in the 'num' array
