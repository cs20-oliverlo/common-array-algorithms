Common Array Algorithms

use the console for all of the functions

My Includes

checks if an array includes a specific things
in the console, type "myIncludes(item from the array below)" make sure to include "" around strings, this will show true if the item is in the array
list: '1', '2', '3', 'flowers', 'genuine', 'Maynards'
(does take capitals into account)

My IndexOf

checks if an array includes a specific things then states where those things are
in the console, type "myIndexOf(item from the array below)" make sure to include "" around strings, this will show the index where the item is at (-1 if the item is not in the array)
list: '1', '2', '3', 'flowers', 'genuine', 'Maynards'
(does take capitals into account)

Average Function Assignment

averages the numbers in the list2 array
type "average()"
list2: '4', '6', '-7', '9', '10'

My Max Assignment

will find the greatest value in the 'list2' array
type "myMax()"
list2: '4', '6', '-7', '9', '10'

Count Function Assignment

will count the number of times and item is found in the 'list3' array
type "count(item from the array below)" make sure to include "" around strings, this will show the number of times the written item is found in the array
list3: '1', '1', '1', '2', '2', '3', '3', '3', '3', '4', 'grass', 'grass', 'lancaster'
